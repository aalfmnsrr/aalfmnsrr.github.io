Designing and Developing a Personal Website Landing Page with HTML and CSS

In this assignment, I designed and developed a personal website landing page using HTML and CSS. This assignment depicted that the importance of web development skills is crucial in ensuring the needs of stakeholders. It inspired me to continue exploring advanced CSS techniques and modern design trends.
